客户端请求 (HTTP Request)
        │
        ▼
┌───────────────────────────┐
│        Controller         │
│ - 接收请求参数            │
│ - 简单校验                │
│ - 调用 Service 方法        │
└───────────▲───────────────┘
            │
            ▼
┌───────────────────────────┐
│         Service           │
│ - 实现核心业务逻辑        │
│   （例如检查用户名是否存在）│
│ - 可以处理事务            │
└───────────▲───────────────┘
            │
            ▼
┌───────────────────────────┐
│         Mapper            │
│ - 继承 BaseMapper<T>      │
│ - 执行 SQL 操作            │
└───────────▲───────────────┘
            │
            ▼
┌───────────────────────────┐
│         数据库 (DB)       │
│ - 存储实体数据            │
│ - 提供查询/增删改支持      │
└───────────────────────────┘

返回结果：数据库 → Mapper → Service → Controller → 客户端
